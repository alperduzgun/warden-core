OverviewGenel Bakis

Warden is a sophisticated AI-powered code analysis and validation system designed to detect security vulnerabilities, code quality issues, and architectural problems across multiple programming languages. Warden, birden fazla programlama dilinde guvenlik aciklari, kod kalitesi sorunlari ve mimari problemleri tespit etmek icin tasarlanmis gelismis bir yapay zeka destekli kod analiz ve dogrulama sistemidir.

Secret ManagementGizli Anahtar Yonetimi

Chain of responsibility pattern for multi-source secret loading. Coklu kaynak gizli anahtar yuklemesi icin sorumluluk zinciri deseni.

Provider Priority

SecretManager
└── Auto-detection chain:
    1. Azure Key Vault (if AZURE_KEY_VAULT_URL set)
    2. GitHub Actions (if GITHUB_ACTIONS=true)
    3. .env file (local development)
    4. Environment variables (fallback)

Usage Example

from warden.secrets import SecretManager, get_secret

# Auto-configured manager
manager = SecretManager()
result = await manager.get_secret("AZURE_OPENAI_API_KEY")

if result.found:
    print(f"Value from {result.source.value}")
    api_key = result.value

# Convenience function
api_key = await get_secret("API_KEY")

Semantic Search (Qdrant)Semantik Arama (Qdrant)

Vector database integration for intelligent code similarity search. Akilli kod benzerlik aramasi icin vektor veritabani entegrasyonu.

Components

CodeIndexer

Indexes code chunks with vector embeddings into Qdrant collections.

SemanticSearcher

Performs vector similarity search with filtering support.

EmbeddingGenerator

Generates embeddings using OpenAI/Azure OpenAI models.

ContextRetriever

Retrieves relevant code context for LLM analysis.

Workflow

1. Code Discovery → Discover all files
2. AST Parsing    → Extract functions, classes, modules
3. Code Chunking  → Split by ChunkType (function, class, module)
4. Embedding      → Generate vectors (384-768 dimensions)
5. Indexing       → Store in Qdrant with metadata
6. Search         → Vector similarity + filters

Configuration

QDRANT_URL=http://localhost:6333
QDRANT_API_KEY=your-api-key  # for cloud
QDRANT_COLLECTION=warden_code_chunks
